# これなに

最近github pagesで静的サイトを運営しているんですが、  
些細な更新等で一々コミットするのが面倒くさくなりまして  
ならば、自動化してしまえと片手間で書いたものです。

大前提として、
Remote先がすでに登録されている事  

## 新機能
コミットメッセージを入力できるようにしました。

## 使い方
コンパイルしたファイルを/usr/bin/下に配置すると使えます。  

``` git remote -v ```でpush先が確定されている  
その状態でこのプログラムを走らせると、  
``` Commit Time: 年/月/日 時:分 入力したコミットメッセージ ```

以上のコミットメッセージでコミットされます。
